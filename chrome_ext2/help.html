G-Recorder Help

This Chrome extension is intended as an aide for those who need to search Google maps. It helps you to capture (scrape) results from the browser and creates quick search links to several websites where researchers are usually getting additional information about businesses and places. It creates a tab in your browser where you can work with results. Please leave comments on how you would like this software to be improved in the future. This software is provided free of charge and on as-is basis. Please contact me if you feel like making a small donation to the author. Thanks and enjoy!

1. Delete data from the app.

2. Run Google maps search.

3. Scroll through all the results, using next button on Google results.

4. Download results as a CSV.

5. Or open results in a new tab. Results are not updated automatically on this page, you need to reload the tab after each search.

6. Copy “name”, “phone”, or “address” by clicking corresponding text.

7. Copy “website” by clicking copy icon.

8. Utilize “quick search” links for Hunter, Indeed, or LinkedIn.

9. Highlight or un-highlight the row by clicking the highlight icon.

10. Use Options to setup configuration for search results. You can configure any of the “quick search” icons. Results, which don’t have websites associated with them can be excluded. Companies with multiple locations can be mentioned ones or multiple times.
